# Snag
Downloads all files from a simple HTTP/HTTPS file server
Requirements:
- aria2
- Python 3.14
- Mechanical Soup

Installation:
- Virtual environment:
  - `python -m venv venv`
  - enter venv
  - `pip install -r requirements.txt`

Or just install Mechanical Soup via your package manager

Known issues:
- currently flattens the directory structure always
